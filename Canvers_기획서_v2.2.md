# SEbit Canvers 기획서 v2.2

## 서비스 정의
Canvers는 업종과 디자인 테마를 선택하고 간단한 정보를 입력하면, AI가 맞춤형 홈페이지 시안을 생성하고 즉시 수정 가능한 CMS를 제공하는 서비스입니다.

> **한 줄 정의:** 회원가입 없이 3분 만에 홈페이지 시안과 CMS를 만드는 서비스

---

# 1. 핵심 방향성
- 완성형 테마 + CMS 중심 MVP
- URL 기반 스타일 분석은 v2 기능으로 이관
- 회원가입 없는 무료 서비스로 빠른 사용자 확보
- AI 카피 생성 및 홈페이지 시안 자동 생성

# 2. 타겟 고객
- 소상공인
- 1인 사업자
- 스타트업
- 병원 및 전문 서비스 업체
- 카페 및 음식점
- 미용 및 뷰티 업종

# 3. 서비스 흐름
메인 진입 → 업종 선택 → 디자인 테마 선택 → 가게 정보 입력 → AI 시안 생성 → 시안 미리보기 → CMS 즉시 발급 → 사이트 운영

예시:

# 10. 구현 및 업데이트 내역 (2026-07-05)

## 메인 화면 디자인
- 기존 Hero의 큰 타이포그래피와 정보 위계는 유지
- Soft Pastel 시안의 보라색 포인트, 그라디언트, 둥근 카드, 그림자 스타일 적용
- Canvers 브랜드명과 무료·비회원 시작 메시지 반영
- 데스크톱 및 모바일 반응형 레이아웃 적용

## 사용자 흐름
- URL 기반 스타일 분석 진입 제거(v2 기능으로 이관)
- `업종 선택 → 디자인 테마 선택 → 가게 정보 입력 → AI 시안 생성` 흐름으로 변경
- 메인 화면에서 선택한 업종과 테마를 정보 입력 화면으로 전달
- 업종 또는 테마를 선택하지 않은 경우 안내 메시지 표시

## 업종 및 테마
- 업종 6종 제공: 카페·베이커리, 뷰티·살롱, 피트니스, 클리닉, 레스토랑, 온라인 스토어
- 디자인 테마 4종 제공: Minimal, Editorial, Bold, Soft
- 업종별 About 및 주요 제공 항목용 기본 AI 카피 확장
- 각 테마별 색상, 폰트, 레이아웃, 여백 및 모서리 스타일 정의

## 정보 입력 및 생성
- 업종과 테마를 확인·변경할 수 있는 정보 입력 화면 구성
- 가게명, 사이트 주소, 한 줄 소개, 핵심 제공 항목, 연락처, 주소, 영업시간 입력 지원
- AI 시안 생성 액션과 기존 미리보기·CMS 흐름 연결 유지
- 서비스 메타데이터를 최신 서비스 정의에 맞게 변경

## 개발 및 검증
- Next.js 프로덕션 빌드 성공
- TypeScript 타입 검사 및 정적 페이지 생성 성공
- 로컬 미리보기에서 메인 화면 렌더링 확인
